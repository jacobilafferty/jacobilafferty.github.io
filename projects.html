Algorithmic Market Simulation

Java • Algorithm Design • St. Lawrence University

Oct 2025 - Nov 2025

Engineered a high-frequency trading engine that simulates a stock market limit order book. The system processes thousands of buy/sell orders in real-time, matching opposing agents based on price priority and timestamps.

Key Technical Features
  • Implemented a Dual-Priority Queue system (Min-Heap and Max-Heap) to achieve O(1) access time for the best bid and ask prices.
  • Designed a custom algorithm to calculate the Running Median of transaction prices dynamically without sorting the entire dataset, optimizing time complexity for large data streams.
  • Built a "quality metric" matching logic to prioritize high-value trades while maintaining strict First-In-First-Out (FIFO) ordering for ties.
